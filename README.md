# DONT WORRY

나만의 걱정 고민을 들어주고 도와주는 AI 서비스로, 작성한 고민에 대해 분석하고 도움을 줍니다 <br>
나의 걱정거리들을 기록하고 나중에 봤을때 내가 어떤 걱정을 가지고 있었는지 확인할 수 있는 혁신적인 서비스! 😊

<br>

✅ 홈 화면(소개 페이지) -> 서비스에 대한 전반적인 소개와 이용하는 방법을 볼 수 있습니다.<br/>
✏️ 걱정 작성 페이지 ->  사용자는 걱정거리를 작성하고, AI로부터 위로의 답변을 받을 수 있습니다.<br/>
📥걱정 보관함 -> 사용자가 작성한 내용과 답변은 걱정 보관함에서 다시 확인할 수 있습니다.<br/>
📊걱정 분석(통계페이지) -> 작성한 걱정들을 분석하여, 반복해서 기록한 고민을 도표로 확인할 수 있습니다.<br/>
💌미래의 나에게 보내는 편지(편지 쓰기 페이지) -> 미래의 나에게 보내는 편지를 통해 힘들었던 과거의 자신과 현재의 자신이 얼만큼 달라졌는지 확인해보세요.

### 🗺️ Don't worry - 한줄소개

여러분의 걱정을 담아내고, 따뜻한 말 한마디로 마음을 토닥여주는 작은 친구예요.</br>
고민이 많은 현대인들을 위한, 든든한 지원군의 등장</br>

소개내용<br>

이제 Don't worry 와 함께 자신만의 걱정거리를 이야기해보세요!

---

### 🚀 Don't worry 가 제공하는 주요 기능</br>

**기능1** </br>
✅ 걱정 주제와 감정을 기반으로 모든 고민거리에 대한 조언 제공</br>
✅ 작성한 고민들을 걱정 보관함에서 열람 가능</br>

**기능2** </br>
✅ 마이페이지에서 이전에 작성했던 걱정 내역들을 수치상으로 확인 가능</br>

**기능3** </br>
✅ 미래의 나에게 편지를 작성하여 이메일의 형식으로 받아보기 가능</br>

**기능4** </br>
✅ 작성한 걱정들을 기반으로 주/월 단위로 분석하여 도식화한 차트를 제공</br>
✅ 분석한 고민들을 기반으로 위로의 한마디와 조언을 제공</br>
✅ 이전 월/주 대비 많이 감소한 고민과 많이 증가한 고민에 대한 분석 제공</br>
